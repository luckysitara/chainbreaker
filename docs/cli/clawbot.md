# `chainbreaker clawbot`

Legacy alias namespace kept for backwards compatibility.

Current supported alias:

- `chainbreaker clawbot qr` (same behavior as [`chainbreaker qr`](/cli/qr))

## Migration

Prefer modern top-level commands directly:

- `chainbreaker clawbot qr` -> `chainbreaker qr`
