# `chainbreaker setup`

Initialize `~/.chainbreaker/chainbreaker.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
chainbreaker setup
chainbreaker setup --workspace ~/.chainbreaker/workspace
```

To run onboarding via setup:

```bash
chainbreaker setup --wizard
```
