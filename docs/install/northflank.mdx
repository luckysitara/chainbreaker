Deploy Chainbreaker on Northflank with a one-click template and access it through the web Control UI.
This is the easiest "no terminal on the server" path: Northflank runs the Gateway for you.

## How to get started

1. Click [Deploy Chainbreaker](https://northflank.com/stacks/deploy-chainbreaker) to open the template.
2. Create an [account on Northflank](https://app.northflank.com/signup) if you don't already have one.
3. Click **Deploy Chainbreaker now**.
4. Set the required environment variable: `CHAINBREAKER_GATEWAY_TOKEN` (use a strong random value).
5. Click **Deploy stack** to build and run the Chainbreaker template.
6. Wait for the deployment to complete, then click **View resources**.
7. Open the Chainbreaker service.
8. Open the public Chainbreaker URL at `/chainbreaker` and connect using your `CHAINBREAKER_GATEWAY_TOKEN`.

## What you get

- Hosted Chainbreaker Gateway + Control UI
- Persistent storage via Northflank Volume (`/data`) so config/credentials/workspace survive redeploys

## Connect a channel

Use the Control UI at `/chainbreaker` or run `chainbreaker onboard` via SSH for channel setup instructions:
